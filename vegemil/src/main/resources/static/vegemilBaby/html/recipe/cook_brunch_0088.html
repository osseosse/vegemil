    맥앤치즈

    크리스마스에 어울리는 요리, 맥앤치즈입니다^^
    두유를 활용하여 더 건강하고 고소하고 진~한 맥앤치즈~!
    누구나 좋아하는 치즈가 듬~뿍, 비주얼도 근사해서 크리스마스 파티음식으로 딱이겠죠?

    준비재료

    무첨가두유 2팩, 빵가루 30g, 마카로니 2Cup, 체다치즈 4장, 고다치즈 2장, 소금1t, 파마산치즈가루 20g

  • 01

    끓는 물에 소금(1t)을 넣고 마카로니(2C)를 잘 삶는다.

  • 02

    무첨가두유(2팩)에 체다치즈와 고다치즈를 넣고 녹인다.

  • 03

    파마산 치즈가루(20g)를 넣어 잘 섞어준다.

  • 04

    삶아놓은 마카로니와 치즈를 잘 섞은 후 오븐팬에 옮겨 고르게 편다.

  • 05

    빵가루(30g)와 파슬리를 뿌린 후, 180도 오븐에 넣고 15분간 구워주면 완성!
