• 영양간식

    조랭이 두유볶음

    아기자기한 조랭이떡을 고소한 두유에 볶으면 어떨까? 부드러운 식감이 고소한 소스에 잘 어울린다. 새우와 한치를 넣어 더욱 풍성한 맛이 일품이다.

    준비재료

    우리안 콩국물 1컵, 조랭이떡 1줌, 생크림 1/2컵, 칵테일 새우 1컵, 한치 1/2마리, 브로콜리 1/6송이, 다진 마늘 1큰술, 소금, 후추, 오일 약간씩

  • 한치는 껍질을 벗겨 링으로 썰고, 브로콜리는 송이송이 떼어놓고, 조랭이떡은 물에 불려 준비한다.

  • 01

    오일을 두른 팬에 다진 마늘과 조랭이떡을 넣고 볶는다.

  • 02

    한치, 새우, 브로콜리를 넣고 볶는다.

  • 03

    콩국물과 생크림을 부어 끓인 후, 소금, 후추로 간하여 농도를 낸다.

  • Tip
  • 1.한치 대신 오징어로 대체해도 좋다.
    2.한치껍질은 굵은 소금으로 문질러주면 잘 벗겨진다.
