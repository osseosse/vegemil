The Food Safety System Certification 22000

식품안전시스템

국제식품안전협회(GFSI)가 승인한 국제식품규격 중 하나로
세계 각국의 인정기관(Accreditation Bodies)으로 부터 인정되고 있는 규격이며
글로벌 식품기업에서 가장 적극적으로 채택하고 있는 식품안전규격입니다.
